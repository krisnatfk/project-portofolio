EduPath is an online learning mobile application developed using Flutter. The app won **1st Place in Mobile Development** at Expo Teknokrat 2026.

## Features

- 📚 Structured online courses and learning paths
- 🎯 Progress tracking and quiz assessments
- 🔔 Push notifications for reminders
- 🌐 Responsive UI across Android devices

## Awards

🏆 **1st Place – Mobile Programming** at Expo Teknokrat 2026

## Tech Stack

- **Framework**: Flutter (Dart)
- **Backend**: Firebase
- **Database**: Firestore

## Development

```bash
flutter pub get
flutter run
```
