A personal portfolio and blog site showcasing my projects, achievements, certifications, and professional experience as an Informatics student and developer.

## Features

- 🌐 Multi-language support (Indonesian & English)
- 🌙 Dark/Light mode toggle
- 📊 GitHub contribution graph
- 💬 Real-time Chat Room powered by Supabase
- 🏆 Achievements & certificates gallery
- 📁 Project showcase with tech stack details
- 📬 Contact form with email notification

## Tech Stack

- **Frontend**: Next.js 14 (App Router), TypeScript, Tailwind CSS
- **Database**: Supabase (Chat Room)
- **Auth**: NextAuth.js (Google & GitHub OAuth)
- **Deployment**: Vercel

## Getting Started

```bash
git clone https://github.com/krisnatfk/portfolio.git
cd portfolio
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.
