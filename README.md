## 📘 Introduction

This is my personal portfolio website built using modern technologies such as Next.js and TypeScript.

I use this platform to share projects, achievements, activities, and resources. The site is constantly evolving as I add more features and improvements.

Feel free to explore the source code, use it as inspiration, or fork it as a template under the provided license. If you find this project useful, consider giving it a star ⭐.

## 🚀 Features

### 🕗 Wakatime Statistics
Displays live coding statistics from Wakatime using serverless API routes.

### 🗳 Project Showcase
Projects are fetched from GitHub API and displayed with custom thumbnails from Supabase Storage.

### 🌍 Internationalization
Supports English and Indonesian using `next-intl`.

### 📊 Developer Dashboard
Interactive dashboard visualizing GitHub contributions, Wakatime data, and more.

### 🏆 Achievements
Certificates and achievements stored in Supabase with image gallery.

### 📅 Activities
Activities and events displayed with photos from Supabase Storage.

## 🛠 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/krisnatfk/project-portofolio
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Configure Environment Variables

Copy `.env.example` to `.env` and replace with your own credentials.

```bash
cp .env.example .env
```

### 4. Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
